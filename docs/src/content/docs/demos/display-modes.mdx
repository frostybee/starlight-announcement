This page demonstrates how multiple announcements are displayed. The current site uses **stack** mode, showing all announcements vertically.

## Current Mode: Stack

Look at the top of this page - you'll see multiple announcements stacked vertically. This is the default `stack` display mode.

## Display Mode Comparison

### Stack Mode (Current)

All active announcements are displayed one below the other:

```js
starlightAnnouncement({
  displayMode: 'stack', // Default
  announcements: [
    { id: 'first', content: 'First announcement', variant: 'note' },
    { id: 'second', content: 'Second announcement', variant: 'tip' },
    { id: 'third', content: 'Third announcement', variant: 'caution' },
  ],
})
```

**Result:** All three announcements visible, stacked vertically.

### First Mode

Only the first matching announcement is shown:

```js
starlightAnnouncement({
  displayMode: 'first',
  announcements: [
    { id: 'priority-1', content: 'High priority', variant: 'danger' },
    { id: 'priority-2', content: 'Medium priority', variant: 'caution' },
    { id: 'priority-3', content: 'Low priority', variant: 'note' },
  ],
})
```

**Result:** Only "High priority" announcement visible.

### Rotate Mode

Announcements cycle through automatically:

```js
starlightAnnouncement({
  displayMode: 'rotate',
  rotateInterval: 5000, // 5 seconds
  announcements: [
    { id: 'promo-1', content: 'First promotion', variant: 'tip' },
    { id: 'promo-2', content: 'Second promotion', variant: 'note' },
    { id: 'promo-3', content: 'Third promotion', variant: 'tip' },
  ],
})
```

**Result:** Announcements cycle every 5 seconds.

## Configuration

The display mode is set at the plugin level:

```js
starlightAnnouncement({
  displayMode: 'stack', // 'stack' | 'first' | 'rotate'
  rotateInterval: 5000, // Only used with 'rotate' mode
  announcements: [/* ... */],
})
```

## Mode Selection Guide

| Mode | Best For |
|------|----------|
| `stack` | Few important announcements that all need visibility |
| `first` | Priority-based announcements where order matters |
| `rotate` | Promotional content or many announcements |

```js
{
  id: 'demo-stack-3',
  content: 'Third stacked announcement (caution variant)',
  variant: 'caution',
  showOn: ['/demos/display-modes/'],

```

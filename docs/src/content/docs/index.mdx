## Features

<CardGrid>
  <Card title="Dismissible Banners" icon="close">
    Users can dismiss announcements with a close button. Dismissed state persists in localStorage across sessions.
  </Card>
  <Card title="Multiple Variants" icon="information">
    Choose from `note`, `tip`, `caution`, or `danger` variants matching Starlight's aside styles.
  </Card>
  <Card title="Date Scheduling" icon="seti:clock">
    Show announcements only during specific date ranges with `startDate` and `endDate` options.
  </Card>
  <Card title="Page Targeting" icon="document">
    Use glob patterns with `showOn` and `hideOn` to control which pages display each announcement.
  </Card>
  <Card title="Display Modes" icon="bars">
    Stack multiple announcements, show only the first match, or rotate through them with smooth transitions.
  </Card>
  <Card title="Internationalization" icon="translate">
    Built-in translations for multiple languages plus support for localized announcement content.
  </Card>
  <Card title="Rich Content" icon="pencil">
    Use HTML in announcement content for bold text, links, and other formatting.
  </Card>
  <Card title="CSS Customization" icon="seti:css">
    Fully customizable via CSS variables and selectors. Create custom color themes with ease.
  </Card>
</CardGrid>
